    Was Macht Das Parlament?!

    Auf dieser Webseite sollen die verschiedenen Vorgänge des Berlnier Parlaments anschaulich dargestellt werden. Die Daten, die hier angeboten werden stammen alle vom Webafutritt des Berliner parlaments Links kannst du zum Beispiel Listen von Parteien, Abgeordneten oder Schlagworten sehen, oder durch den Feed der Arbeit scrollen. Oder probier doch mal die Suche rechts oben?

    Du hast einen Fehler entdeckt? Ist etwas offensichtliches nicht gut aufbearbeitet? Willst du gar bei dem Projekt mitmachen? Nicht ärgern - sondern aktiv werden: Lass einfach weiter unten eine Nachricht da!

    Technisches

    Du willst wissen wie diese Seite entsteht? Bitteschön! Die Schritte unten werden jede Nacht ausgeführt, zuletzt am 26.10.2022 um 01:51 Uhr. Damit sind hier stets die neuesten Entwicklungen zu finden. Orchestriert wird das alles mit Hilfe von den guten alten make und cron.

    1. Die Daten, die auf dieser Seite bereitgestellt werden, stammen von hier.
    2. Dazu gibt es noch eine zusätzliche, von Hand geschriebene Datei mit Infos über Parteien und Abgeordnente.
    3. Mit wenigen cleveren Zeilen Code in python wird das XML gelesen und verarbeitet. Ausgespuckt werden markdown Dateien mit dem Inhalt den du hier siehst.
    4. mkdocs baut daraus diesen Hochglanzinternetauftritt im HTML Format im mkdocs-material Theme.
    5. Dannach werden die Dateien nach nach GitHub per git geschoben.
